Precision Spin Coater with Embedded PI Controller

Context: Embedded Control · Microfabrication Lab · University of Tabriz
Embedded C Real-Time Control Systems

Firmware for a high-precision spin coater used in microfabrication and thin-film deposition processes. The system implements closed-loop motor speed control using a Proportional-Integral (PI) controller, ensuring accurate and repeatable coating thickness across substrate runs.

PI Controller

The PI controller reads tachometer feedback at a fixed sampling interval and computes the control output to minimize speed error. The integral term eliminates steady-state offset caused by friction and load variation, while the proportional term provides fast response to speed disturbances. Controller gains were tuned empirically to achieve minimal overshoot and settling time.

Soft-Start Safety Mechanism

A soft-start ramp was implemented to gradually increase motor speed from rest to the target RPM, preventing mechanical shock and reducing stress on both the motor and the substrate. The ramp profile is configurable via firmware parameters and includes an emergency stop interrupt for operator safety.

Real-Time Embedded Loop

The control loop runs on a bare-metal AVR/STM32 microcontroller with deterministic timing enforced by a hardware timer interrupt. All control computation occurs within the interrupt service routine (ISR), guaranteeing a fixed sampling rate regardless of other firmware activity.

Stack: Embedded C · AVR / STM32 · PI Controller · Hardware Timer ISR · Real-Time Control
