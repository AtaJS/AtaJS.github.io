Healthcare MCP Assistant with Multi-Tool Orchestration

Context: Claude API · MCP · Healthcare AI
Claude Sonnet MCP Function Calling

A healthcare query assistant powered by Claude's API that intelligently orchestrates multiple data sources in a single response using function calling. Rather than answering from a single tool, the assistant identifies which combination of tools is needed and synthesizes a unified, accurate answer.

Multi-Tool Orchestration

The assistant has access to four healthcare tools: FAQ lookup, appointment details, lab results, and doctor information. For a query like "Is appointment APT-101 confirmed and what are your office hours?", Claude automatically calls both the appointment lookup and FAQ tools, then synthesizes the results into a single coherent response — rather than answering each part separately.

Evaluation with DeepEval

The system was evaluated using DeepEval across five test cases, measuring completeness of responses and tool selection accuracy. Results: 80% completeness pass rate (4/5 tests) and 100% tool selection accuracy — the model always called the right tools for the query.
