IoT-Integrated Remote Photoplethysmography System

Context: Master's Thesis · University of Oulu, CMVS
Domain: rPPG · IoT · Embedded Systems
rPPG IoT Embedded OpenCV

A complete end-to-end contactless heart rate monitoring platform built as part of the Master's thesis research at CMVS, University of Oulu. Three interconnected devices — a PC, Raspberry Pi 400, and Arduino Mega — form a real-time rPPG measurement system with user authentication, environmental control, and a live signal processing pipeline.

Signal Processing Algorithms

The system implements three established rPPG algorithms: GREEN (simple green channel extraction), POS (Plane-Orthogonal-to-Skin), and CHROM (Chrominance-based method). Each algorithm extracts the blood volume pulse signal from facial video, enabling contactless heart rate estimation without any physical contact with the subject.

System Architecture

The Arduino Mega manages environmental sensors and actuators, communicating with the Raspberry Pi over serial. The Raspberry Pi handles face detection and signal acquisition via camera. The PC runs the deep learning inference pipeline, ensemble methods, and displays real-time heart rate estimates. Devices communicate over a local IoT network using MQTT and HTTP.

Stack: Arduino C/C++ · Python · ESP8266 · Raspberry Pi · OpenCV · MediaPipe · MQTT
