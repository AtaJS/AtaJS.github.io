Explainable Emotion Classification Using Large Language Models

Context: Affective Computing · NLP · University of Oulu, 2025
Few-Shot CoT Prompting LLM Eval

A comparative evaluation of large language models for emotion classification from text, focusing on both accuracy and interpretability. The study assessed multiple LLM architectures under different few-shot learning configurations to understand how model scale and prompt design affect affective computing tasks.

Models Evaluated

Three model families were evaluated: Mistral 7B, Qwen2.5 7B, and FLAN-T5 (Base, 250M parameters). Models were run locally via Ollama for the 7B models and via HuggingFace for FLAN-T5. The surprising key finding was that FLAN-T5 Base (250M parameters) consistently outperformed both 7B models on the emotion classification benchmark.

Few-Shot Configurations

Each model was evaluated under three prompting regimes: zero-shot (no examples), 3-shot (three labeled examples in context), and 5-shot (five examples). Performance was measured across all six basic emotion categories. The study showed that smaller, instruction-tuned models can outperform larger general-purpose models on narrow classification tasks.

Chain-of-Thought Prompting

Chain-of-Thought (CoT) prompting was applied to elicit step-by-step reasoning from the models before producing a final emotion label. This improved both accuracy and interpretability, with the model's reasoning chain providing a human-readable explanation for each prediction.
